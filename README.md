# Hidden Lyfestyle — Two-Page Website

## Pages
- `index.html` — homepage, services, selected work
- `about.html` — brand story and contact form

## Add photos
Place these files in `assets/images/`:
- `hero.jpg`
- `look-01.jpg`
- `look-02.jpg`
- `look-03.jpg`
- `look-04.jpg`
- `look-05.jpg`
- `look-06.jpg`

## Before publishing
Change `booking@hiddenlyfestyle.com` in `script.js` to the client's real booking email.

## GitHub Pages
Upload the contents of this folder to a GitHub repository and enable Pages from the repository settings. Keep `index.html` in the repository root.
